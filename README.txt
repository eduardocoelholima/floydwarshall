** Implementação do Algoritmo Floyd-Warshall

Calcula menores caminhos entre dois vértices quaisquer, dado um grafo sem ciclos negativos.

- A entrada é realizada através de um arquivo texto especificamente formatado.

Este projeto foi desenvolvido durante a Disciplina de Algoritmos e Estruturas de Dados no
Departamento de Informática e Matemática Aplicada da UFRN,
ministrada pelo Professor Bruno Motta de Carvalho.
